# User Stories

## 1. User Management

**US-01:** As an **administrator**, I want to **create, update, deactivate, and delete user accounts**, so that **all user information is accurate, up-to-date, and compliant with company policies**.

**US-02:** As an **administrator**, I want to **assign roles to users and view all activities of my employees** (tasks, leave requests, project contributions), so that **I can manage responsibilities and monitor performance effectively**.

**US-03:** As a **user**, I want to **log in securely to my unique department dashboard**, so that **I can access my tasks, projects, and personal information relevant to my role without seeing unauthorized data**.

## 2. Role Management

**US-04:** As an **administrator**, I want to **define, assign, and manage user roles** so that **users have appropriate access to system features according to their responsibilities**.

**US-05:** As a **user**, I want to **see only the features allowed for my role**, so that **I donâ€™t access restricted areas or perform unauthorized actions**.

## 3. Department Management

**US-06:** As an **administrator**, I want to **create, update, and delete departments**, so that **the organizational structure reflects the current company setup**.

**US-07:** As a **department head**, I want to **view, add, edit, and remove members of my department**, so that **I can manage my team efficiently and ensure each employee has correct access and responsibilities**.

## 4. Project Management

**US-08:** As a **department head**, I want to **create, update, and close projects**, so that **all projects are organized and tracked efficiently**.

**US-09:** As a **department head**, I want to **associate projects with one or more departments**, so that **all relevant teams can collaborate and contribute to the project**.

**US-10:** As a **user**, I want to **view projects assigned to my department(s) and role**, so that **I can focus on relevant tasks without confusion from unrelated projects**.

## 5. Task Management

**US-11:** As a **department head**, I want to **create and assign tasks to employees**, so that **work is clearly distributed and responsibilities are defined**.

**US-12:** As an **employee**, I want to **update the status of my tasks** (e.g., in-progress, completed), so that **my manager can track progress in real-time**.

**US-13:** As a **department head or employee**, I want to **set task priorities and deadlines**, so that **urgent work is prioritized and projects are delivered on time**.

## 6. Progress Tracking

**US-14:** As a **department head**, I want to **view task progress for my employees in real-time**, so that **I can identify bottlenecks and provide support when needed**.

**US-15:** As a **management user**, I want to **see project progress indicators and overall team performance**, so that **I can make data-driven decisions and report to executives**.

## 7. Notifications

**US-16:** As a **user**, I want to **receive notifications when tasks are assigned or updated**, so that **I am always aware of my responsibilities and deadlines**.

**US-17:** As a **department head**, I want to **receive alerts when tasks are completed or delayed**, so that **I can take timely actions to keep projects on track**.

## 8. Reporting

**US-18:** As a **management user**, I want to **generate reports on project status, task completion, and team performance**, so that **I can analyze productivity and make informed decisions**.

**US-19:** As a **department head**, I want to **export reports for my department** (tasks, projects, leave, progress), so that **I can share insights with executives or clients**.
