## 1. User Management

**US-01:** As an **administrator**, I want to **create, update, deactivate, and delete user accounts**, so that **all user information is accurate and up-to-date**.

**US-02:** As an **administrator**, I want to **assign roles to users and view all activities of my employees** (tasks, leave requests, project contributions), so that **I can manage responsibilities and monitor performance effectively**.

**US-03:** As a **user**, I want to **log in securely to my unique department dashboard**, so that **I can access my tasks, projects, and personal information relevant to my role**.

## 2. Role Management

## 3. Department Management

**US-06:** As an **administrator**, I want to **create, update, and delete departments** so that **the organizational structure is accurate**.

**US-07:** As a **department head**, I want to **view, add, edit, and remove members of my department** so that **I can manage my team effectively**.

## 4. Project Management

**US-08:** As a **department head**, I want to **create, update, and close projects** so that **projects are managed efficiently**.

**US-09:** As a **department head**, I want to **associate projects with one or more departments** so that **all relevant teams can work on them**.

**US-10:** As a **user**, I want to **view projects based on my department(s) and role** so that **I see only relevant projects**.
